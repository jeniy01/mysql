DDL(Data Definition Language) 명령 : 데이터 정의어
- 테이블 또는 뷰, 인덱스, 시퀀스 에 대한 생성, 변경, 제거 명령
	Create : 테이블 생성
	create table 테이블명(컬럼명1  타입(크기) [제약조건] ,...);
	create view 뷰명 as 검색명령;

	Drop : 테이블 제거
	drop table 테이블명;
	drop view 뷰명;

	Alter : 테이블 구조 변경
	컬럼 타입 변경 : alter table 테이블명 modify 컬럼명  타입(크기);
	컬럼 추가 : alter table 테이블명 add 컬럼명 타입(크기);
	컬럼 제거 : alter table 테이블명 drop column 컬럼명;
	컬럼명 변경 : alter table 테이블명 rename column 원래컬럼명 to 바꿀컬럼명;

	Rename : 테이블이름 변경
	rename  원본테이블명  to  변경테이블명;

	Truncate : 테이블 구조는 남긴 채 데이터만 삭제
	truncate table 테이블명;

DML(Data Minipulation Language) : 데이터 조작어 = CRUD
해당 테이블에 튜플(레코드)을 검색, 삽입, 삭제, 변경 명령

	Create : 튜플(레코드) 삽입
	insert into 테이블명(컬럼명1, 컬럼명2,...) values (값1, 값2,...)

	Read : 튜플(레코드) 검색
	select [컬럼명1, 컬럼명2,...] from 테이블명 [where 조건절]

	Update
	update 테이블명 set 컬럼명=바꿀값 [where 조건절]

	Delete
	delete from 테이블명 [where 조건절]


DCL(Data Control Language) : 데이터 제어어
사용자 또는 권한, 트랜잭션 등을 제어하는 명령

	사용자 계정 추가
	create user 계정명 identified by 비밀번호;

	사용자 계정 제거
	drop user 계정명;

	비밀번호 변경
	alter user 계정명 identified by 비밀번호;

	접속 계정 변경
	conn  계정명/비밀번호

	권한 부여
	grant connect, resource to 계정명;

	권한 회수
	revoke connect, resource from 계정명;
